# AR 眼鏡去背圖片

此資料夾存放 AR 試戴功能使用的眼鏡正面去背圖片。

## 規格

- 尺寸：1000 × 1000 px（正方形）
- 格式：PNG，透明背景（Alpha 通道）
- 視角：正面
- 數量：35 張

## 命名規則

```
{商品編號}_{品牌}_{系列}_{款式}_{顏色}.png
```

範例：`11644_kitty_round_black.png`

## 使用方式

透過後端 API `/api/glasses/catalog` 取得圖片 URL，AR 頁面以 Three.js 將圖片貼至 3D 平面，搭配程序化鏡腳即時渲染在使用者臉上。
