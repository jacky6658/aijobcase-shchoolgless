# 眼鏡圖片資料夾

## 資料夾對照（臉型）

| 資料夾 | 臉型 |
|--------|------|
| `oval/` | 蛋形臉 |
| `long/` | 長形臉 |
| `heart/` | 倒三角臉 |
| `square/` | 方形臉 |
| `round/` | 圓形臉 |
| `diamond/` | 菱形臉 |

## 檔案命名規則

```
{臉型}_{框型}_{商品ID}_{序號}.jpg
```

範例：`oval_boston_10678_01.jpg`

## 框型代碼對照

| 代碼 | 框型 |
|------|------|
| `boston` | 波士頓框 |
| `butterfly` | 蝴蝶框 |
| `round_lg` | 大圓框 |
| `square` | 正方框 |
| `cateye` | 貓眼框 |
| `round` | 圓框 |
| `oval_fr` | 卵框 |
| `aviator` | 飛行員 |

## 圖片引用方式

```
https://raw.githubusercontent.com/jacky6658/aijobcase-shchoolgless/main/images/glasses/{臉型資料夾}/{檔名}.jpg
```
